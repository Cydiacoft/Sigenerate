# 道路标志覆盖清单（GB 5768 / Wikimedia）

> 数据来源参考：  
> - https://commons.wikimedia.org/wiki/Road_signs_in_China  
> - 项目内 `lib/signs/gb5768_signs.dart` 与 `assets/road_signs/`

## 当前状态（2026-03-27）

- 道路端已接入：`48` 个 SVG 资产
- 覆盖范围：禁令/警告/指示/信息四大类常见高频标志
- 验证状态：`flutter analyze --no-pub` 通过

## 已实现（按功能分组）

### 禁令（已实现）

- 停车让行、减速让行
- 禁止驶入、禁止通行
- 禁止机动车/货车/摩托车/行人进入
- 禁止左转/右转/直行/掉头
- 禁止停车、禁止长时停车
- 禁止鸣喇叭
- 禁止超车、解除禁止超车
- 限速 40、限速 60
- 限宽 2.5m、限高 3.5m、限质量 20t、限轴重 10t

### 警告（已实现）

- 十字交叉、T 形交叉
- 注意行人、注意儿童
- 施工、注意信号灯
- 向左急弯、向右急弯
- 路面湿滑
- 村庄、隧道

### 指示（已实现）

- 直行、左转、右转
- 直行+左转、直行+右转
- 靠左侧道路行驶、靠右侧道路行驶
- 环岛行驶
- 步行

### 指路/信息（已实现）

- 停车场、医院
- 高速公路、旅游区、服务区
